# WL — WebLoader

GUI-приложение для скачивания медиаконтента (видео и аудио) с YouTube и сотен других сайтов в многопоточном режиме с максимальной скоростью.


## Установка:

### Шаг 1: Клонируйте репозиторий

Откройте командную строку (cmd) или PowerShell и выполните:

```bash
git clone https://github.com/motopers/WL.git
&& cd WL
```

### Шаг 2: Запуск

Из командной строки, выполните:


```bash
WebLoader.lnk
```
Или напрямую **Release (готовая сборка):**
```bash
cd Release
.\WebLoader.exe
```


## Возможности

- Скачивание видео и аудио медеаконтест с интернет сайтов
- Многопоточная загрузка через aria2c
- Выбор качества видео (Лучшее, 1080p, 720p, 480p, 360p)
- Визуальный прогресс-бар
- Кнопка отмены загрузки
- Режим "Лучшее" — скачивает лучшее доступное качество максимально кодека (Webm, AV1)
- Режимы 1080p/720p/480p/360p — H.264 кодек


## Зависимости

- yt-dlp — ядро загрузки
- aria2c — многопоточный загрузчик, разпараллеливает загрузку чанков
- deno — JavaScript runtime, для yt-dlp
- .NET 8.0 — платформа для GUI (Windows Forms, C#)
